##Como testar o modelo localmente
# 1. Instale as dependências
# 2. Execute o script RF.py
# 3. Verifique o arquivo rf_ia_model.pkl gerado
# 4. Converta o modelo para ONNX usando skl2onnx
# 5. Verifique o arquivo rf_ia_model.onnx gerado
# 6. Execute o script RunConvertedModel.py para testar a inferência
# 7. Adicione o arquivo .onnx ao projeto Next.js na pasta public

## Como Treinar o modelo
# 1. Siga todos os passos acima, mas antes adicione os dados que devem treinar o modelo no data.json
